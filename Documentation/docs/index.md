# Overview

Breeze is an external, flexible Asset Manager.

It focuses on 3D Animation and opensource software.
It is currently being built with Blender in mind.
later it will be easily extendable to Krita, Guerilla Render, or any other Software.

<div class="grid cards" markdown>


- :material-home-import-outline:  __External__

    ___ 
    
    Breeze is not contained within a software.

    It manages files from the outside, and can work with multiple software simultaneously.


- :material-axis-arrow:  __Flexible__
    
    ___
    
    Asset outputs can be cast as ingredient, from anywhere to everywhere.
    
    Those ingredients are auto-set, or prefiltered to match the context.
    But the base setups can be bypassed whenever needed.


- :material-check-underline-circle:  __Consistent__
    
    ___
    
    - If a shortcut exists somewhere, it also works elsewhere
    - A right click anywhere in a popup will cancel it and close it

</div>
