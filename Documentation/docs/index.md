# Overview

Breeze is and flexible and agnostic Asset Manager. It focuses on 3D Animation and open source software.

It is currently being built with Blender in mind.
Later it will be extended to more software such as Krita, Guerilla Render, etc.

<div class="grid cards" markdown>

    ___

- :material-home-import-outline:  __Agnostic__

    ___
